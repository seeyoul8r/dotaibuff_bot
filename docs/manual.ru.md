# DotAIBuffBot

## Начало работы

- Отправьте боту команду `/start`, чтобы зарегистрироваться.
- Бот покажет приветствие с кратким описанием сервиса и откроет главное меню.
- Если отправить сообщение или нажать старую кнопку до регистрации, бот объяснит назначение сервиса и предложит использовать `/start`.
- Повторная команда `/start` открывает главное меню без повторной регистрации.

## GSI config

- Нажмите `/start` в боте.
- Нажмите кнопку `GSI-конфиг`.
- Нажмите `Получить GSI config` в подменю GSI только если нужно скачать или заменить файл config.
- Скачайте файл `gamestate_integration_dot_ai_buff_bot.cfg`.
- Положите файл в папку Dota 2:
  `...\Steam\steamapps\common\dota 2 beta\game\dota\cfg\gamestate_integration\`
- В Linux используйте:
  `~/.steam/steam/steamapps/common/dota 2 beta/game/dota/cfg/gamestate_integration/`
- В Steam откройте свойства Dota 2 и добавьте в Launch Options: `-gamestateintegration`.
- Запустите или перезапустите Dota 2.
- Если config уже был установлен раньше, скачайте его заново и замените старый файл.
- Когда начнется матч и бот получит первый GSI snapshot с match id, он отправит уведомление с кнопками `♿️ Где враги?` и `📦 Получить совет ИИ`.
- Кнопка `Что такое GSI-конфиг?` в подменю GSI покажет краткую инструкцию по установке.
- Кнопка `Назад` вернет главное меню.

## Рекомендация ИИ

- Нажмите кнопку `📦 Получить совет ИИ` в уведомлении о старте матча.
- Бот объединит накопленное состояние матча с актуальными данными OpenDota и запросит рекомендацию у ИИ.
- После определения полного состава врагов рекомендация учитывает, где и когда врагов последний раз видели на миникарте.
- Пока ИИ формирует ответ, Telegram показывает временный анимированный статус «Думает…».
- Ответ придет одним сообщением: макрогейминг, сборка и текущий приоритет по микрогеймингу.
- В конце ответа бот покажет, когда будет доступен следующий запрос, и добавит кнопки `♿️ Где враги?` и `📦 Получить совет ИИ`.
- Если ИИ не ответит, бот сообщит об ошибке.
- Рекомендацию можно запрашивать не чаще одного раза в 3 минуты для каждого пользователя.
- Если GSI-данные текущего матча еще не получены, бот сообщит об этом без запуска запроса к ИИ.

- В каждом сообщении с рекомендацией бот показывает время матча, на которое она актуальна.
- Нажмите `♿️ Где враги?`, чтобы получить отдельное сообщение с временем матча и блоком `Враги на карте`.
- Блок врагов показывает зоны карты английскими названиями, например `Radiant Triangle`, `Dire Safe Lane` или `Bot Roshan Pit`.
- Если врага еще не видели на миникарте, бот покажет `Not seen yet`; если координаты слишком далеко от ближайшей размеченной зоны, бот покажет `Unknown area`.
- Когда матч завершается, бот отправляет итог: победитель, счет, герой, уровень, KDA и время матча. После этого рекомендацию по завершенному матчу запросить нельзя.

## Язык

- Нажмите кнопку `Lang: EN`, чтобы переключить интерфейс на английский.
- Выбранный язык сохраняется в настройках пользователя.
